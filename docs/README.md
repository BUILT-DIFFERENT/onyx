# Docs Index

This directory tracks current-state documentation for the Onyx repo.

## Project Plans

- `PLAN.md` (repo root) — canonical project plan: architecture, data model, features, tech stack, roadmap, and current status/priorities.
- Warp plan "Onyx UX Specification" — definitive UX reference for all screens, tools, interactions, and visual design decisions.
- `.sisyphus/plans/editor-feel-refactor.md` — detailed editor-feel refactor plan (three-lane architecture, raw stroke model, smoothing pipeline). **Immediate priority.**
- `.sisyphus/plans/android-foundation.md` — Android technical foundation plan (PDF hardening, UI architecture, library/org, recognition/search, CI, reliability).
- `docs/images/` — NoteWise and Samsung Notes reference screenshots (visual north star).
- `docs/Myscript/` — local MyScript docs mirror (SDK API docs, integration guides, concepts, and references).
