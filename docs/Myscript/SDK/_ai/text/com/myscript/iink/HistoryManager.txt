getUndoRedoStackIndex

public final int getUndoRedoStackIndex ()

Returns the number of changesets performed on the part, since content part
was opened.

Note: this function will wait for pending gestures recognition.
Note: the undo stack is partially purged from time to time to control
memory consumption. The number of possible undo changesets at a given
time is provided by getPossibleUndoCount() , while the total number of
changesets since content part was opened is provided by
getUndoStackIndex() .

Returns:

the number of changesets performed on the part.

Throws:

IllegalStateException [https://docs.oracle.com/en/java/javase/17/docs/api/java.base/java/lang/IllegalStateException.html] - when this history manager is closed.

IllegalStateException [https://docs.oracle.com/en/java/javase/17/docs/api/java.base/java/lang/IllegalStateException.html] - when associated editor has been destroyed.

IllegalStateException [https://docs.oracle.com/en/java/javase/17/docs/api/java.base/java/lang/IllegalStateException.html] - when editor is not associated with a part.

@NotNull
public final @NotNull String [https://docs.oracle.com/en/java/javase/17/docs/api/java.base/java/lang/String.html] getUndoRedoIdAt (int stackIndex)

Returns the id associated with an undo/redo stack index.
Valid stack index values range from (current stack index - possible undo count) to (current stack index + possible redo count - 1).

Note: this function will wait for pending gestures recognition.

Returns:
