Superimposed

All languages from the previous list can be recognized both in cursive and superimposed modes.
Non-interactive languages

While most of the languages in the previous list are compatible with both interactive and non-interactive modes, the following additional languages are restricted to non interactive mode:

    Arabic
    Farsi - Iran
    Hebrew - Israel
    Hindi - India
    Thai - Thailand
    Urdu - Pakistan
    Vertical Japanese - Japan

Emojis

    7 base emojis can be recognized as part of the text, along with a total of 16 variants coming as text candidates:
        вқӨ (variants: рҹ’“, рҹ’ҷ, рҹ’•)
        рҹ’” (variant: рҹ’ҳ)
        рҹҳҗ (variants: рҹҳ’, рҹҳ“)
        рҹҷӮ (variants: рҹҳҒ, рҹҳғ, рҹҳҚ, рҹҳ„, рҹҳҠ)
        рҹҷҒ (variants: рҹҳһ, рҹҳӯ, рҹҳ , рҹҳ©)
        рҹҳү (variant: рҹҳң)
        рҹҷғ

So drawing following emoji will let you choose among the variants by selecting your favorite candidate:

Other notable recognition features

    Circled digits: в‘  в‘Ў в‘ў в‘Ј в‘Ө в‘Ҙ в‘Ұ в‘§ в‘Ё в‘© в‘Ә в‘« в‘¬ в‘ӯ в‘® в‘Ҝ в‘° в‘ұ в‘І в‘і and Circled asterix: вҠӣ

    Checkable items: The following ballot boxes can be recognized: вҳҗ вҳ‘ вҳ’.

Futhermore, either on ink or already converted the ballot box вҳҗ can be checked with the рҹ—ё-type mark or the x mark. The conversion will then produce the вҳ‘ and вҳ’ ballot boxes.

When a ballot box starts a line, tapping on it will check it with the рҹ—ё-type mark too.

    Ballot x: вң— is supported as a bullet.

    Mathematics: вҲҖ, вҲғ, вҲ„, вҲҲ, вҲү, вҹӮ

    Greek: Оұ, ОІ, О”, ОЈ

    Units: ВҪ, в…“, Вј, в…”, Вҫ, mВІ, mВі
